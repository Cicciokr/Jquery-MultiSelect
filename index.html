Jquery 1.8+
Mobile Friendly
Open Source
Customize with CSS

Options

Name type default description
dataSource Array [] Array with generate the select multiple
value string "ValueCombo" Property of datasource with value
label string "TextCombo" Property of datasource with shown text
afterInit function function(container){} Function to call after the multiSelect initilization.
beforeSelect function function(data){} Function to call before one item is selected. It return true or false, if false item not passed to selected
beforeSelectable function function(data){} Function to call before one item is deselected. It return true or false, if false item not passed to deselected
afterSelect function function(values){} Function to call after one item is selected.
afterSelectable function function(values){} Function to call after one item is deselected.
disabledClass String 'disabled' CSS class for disabled items.
dblClick Boolean false Replace the defautl click event to select items by the dblclick one.
cssClass String "" Add a custom CSS class to the multiselect container.
height String "auto" Height of container, default is auto.
buttons Array The button for manage the list
buttonsSelected Array Button to manage selected item

Methods

Name type default description
returnSelected function function(){} Return the selected element (return datasource date), in order as appaear
returnSelectable function function() {} Return the deselected element (return datasource date), in order as appaear
setDatasource function function(dataSource){} Set the new datasource on the combo
setSelectedElement function function (element) { Input datasource element that you want to select. (Take the 'value' of each element to select it)
